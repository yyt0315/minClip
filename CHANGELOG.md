# Changelog

## v1.2.0 - 2026-06-05

- 官网首页重写为更聚焦的产品介绍结构，突出“先查看，再复制”的核心体验
- 首页文案更新，新增普通模式 / 极客模式的使用定位说明，更贴近日常使用场景
- 核心功能区视觉重构，使用统一风格图标替代 emoji，整体观感更稳定
- 调整功能呈现优先级，首页优先展示搜索、预览、文本处理和长期沉淀等高频价值点
- 优化按钮、卡片、背景和留白细节，提升官网在桌面端与移动端的浏览质感

## v1.1.0 - 2026-06-04

- 主界面升级为“侧边栏 + 历史列表 + 预览区”的三栏联动工作台，浏览和整理效率更高
- 新增右侧预览面板，支持更顺手地查看长文本、图片、文件和富文本内容
- 内置文本处理工具，支持 JSON 格式化、去除富文本格式、URL 解码、Base64 编解码等操作
- 支持将网页或文档中的富文本内容转换为更易整理的 Markdown 预览
- 优化键盘工作流，支持 `Space` 快速预览、`Tab` 聚焦搜索、`⌘↑/↓` 快速跳转、`⌘⇧1~0` 直接复制前 10 条内容
- 搜索能力升级，支持拼音全拼、拼音首字母和英文首字母快速匹配
- 偏好设置界面重构，功能分组和导航更清晰
- 底层存储升级为 SQLite + JSON 混合方案，历史记录读写性能和大规模使用稳定性更好
- 新增图片 SHA256 去重机制，重复复制同一图片时不再重复占用磁盘空间
- 增强隐私与稳定性，支持忽略密码管理器等敏感应用内容，并优化数据写入安全性

## v1.0.0 - 2026-05-20

- 首个公开版本
- 支持剪贴板历史记录
- 支持永久保存、分类管理和快捷键呼出
