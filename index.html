MinClip

面向高频复制场景的 macOS 菜单栏剪贴板工具

自动记录剪贴板历史，支持文本、图片、文件和富文本内容。现在提供三栏联动工作台、右侧预览区、拼音搜索和内置文本处理能力。

↓ 下载最新版本 GitHub 更新记录

当前版本 v1.1.0 · 支持 macOS 14+ · Intel & Apple Silicon

界面预览

核心功能

📋

自动记录剪切板历史

自动捕获文本、图片、文件和富文本内容

⚡

菜单栏快速访问

点击菜单栏图标或使用 ⌘⇧V 快捷键打开历史记录

🧭

三栏联动工作台

侧边栏、历史列表和预览区协同工作，浏览与整理更高效

⭐

永久保存收藏

将常用内容永久保存到分类中，支持快捷键快速保存

📌

置顶显示

永久保存的内容可置顶，始终显示在列表最上方

🏷️

分类管理

自定义分类，支持颜色、图标、拖拽排序和独立存储路径

🔍

智能搜索

支持中文、拼音全拼、拼音首字母和英文首字母快速匹配

👁️

右侧预览面板

更适合查看长文本、图片、文件和富文本内容

📤

导出功能

将分类内容导出为 CSV 文件

🔧

内置文本处理工具

支持 JSON 格式化、去除富文本格式、URL 解码、Base64 编解码等操作

📝

富文本转 Markdown

复制网页或文档内容后，更容易二次整理、沉淀和复用

🔒

隐私与稳定性增强

支持忽略密码管理器等敏感应用内容，并优化数据存储安全性
