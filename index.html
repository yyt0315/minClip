MinClip

简洁的 macOS 菜单栏剪切板管理工具

自动记录剪切板历史，支持文本、图片、文件和富文本内容。菜单栏快速访问，永久保存收藏，智能搜索过滤。

↓ 下载最新版本 GitHub

支持 macOS 14+ · Intel & Apple Silicon

界面预览

核心功能

📋

自动记录剪切板历史

自动捕获文本、图片、文件和富文本内容

⚡

菜单栏快速访问

点击菜单栏图标或使用 ⌘⇧V 快捷键打开历史记录

⭐

永久保存收藏

将常用内容永久保存到分类中，支持快捷键快速保存

📌

置顶显示

永久保存的内容可置顶，始终显示在列表最上方

🏷️

分类管理

自定义分类，支持颜色、图标、拖拽排序和独立存储路径

🔍

智能搜索

实时搜索过滤，支持拼音首字母缩写匹配

👁️

悬停预览

鼠标悬停查看文本/图片/文件的详细内容

📤

导出功能

将分类内容导出为 CSV 文件

🔧

文本加工工具

内置 JSON 格式化、URL 解码、Base64 编解码等工具

🔒

隐私安全

自动检测并忽略密码管理器等敏感应用的剪切板内容
